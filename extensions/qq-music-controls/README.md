# QQ Music Controls

Control the QQ Music macOS app from Raycast.

## Features

Controls:
- play
- pause
- next track
- previous track
- volume up
- volume down
- like currently playing track
- toggle lyrics display
- play modes:
  - shuffle
  - sequencial
  - repeat
